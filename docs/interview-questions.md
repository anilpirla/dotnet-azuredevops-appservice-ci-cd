# Interview Questions

## What is CI/CD?

CI/CD is an automated process that continuously integrates code changes and deploys applications.

---

## Why use Azure DevOps?

Azure DevOps provides:

- Source Control
- CI/CD Pipelines
- Boards
- Test Management
- Artifact Management

---

## What is YAML Pipeline?

YAML pipeline is pipeline-as-code used to automate build and deployment.

---

## Why use Azure App Service?

Azure App Service is a Platform as a Service offering for hosting web applications.

---

## Explain your project flow.

Developer pushes code to GitHub.

Azure DevOps triggers build.

Application builds.

Artifact generated.

Deploy stage downloads artifact.

Application deployed to Azure App Service.

---

## What is Service Connection?

Service Connection securely connects Azure DevOps with Azure resources.
