# Prerequisites

Before running this project ensure the following requirements are available.

## Azure Requirements

- Azure Subscription
- Resource Group
- Azure App Service
- Azure DevOps Organization

## Azure Resources

Resource Group:
dotnet-rg

Web App:
dotnet-webapp

Runtime:
.NET v10

## DevOps Requirements

- Azure DevOps Project
- Service Connection
- YAML pipeline access

## Local Machine Requirements

- Visual Studio 2022
- .NET SDK
- Git
- GitHub account

Verify installation:

```bash
dotnet --version
git --version
```
