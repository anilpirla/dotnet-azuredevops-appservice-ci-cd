# Project Overview

## Project Name
.NET Application CI/CD Deployment using Azure DevOps and Azure App Service

## Description

This project demonstrates an end-to-end CI/CD implementation for an ASP.NET Core web application using Azure DevOps YAML pipelines and Azure App Service.

The workflow automates:

- Source code integration
- Build and package creation
- Artifact publishing
- Continuous deployment
- Automated release to Azure App Service

## Architecture

Developer
↓
GitHub Repository
↓
Azure DevOps YAML Pipeline
↓
Build + Publish Artifact
↓
Deploy Stage
↓
Azure App Service
↓
Live Application
